What I do

I help organizations make smarter decisions through data. From cleaning messy datasets to building interactive dashboards, I turn numbers into clear, actionable insights using tools like Python, SQL, Tableau and PowerBI.

Data Cleaning & Transformation

I clean, filter, and structure messy or incomplete datasets using Python (Pandas), Excel, PowerBI (Transformation) and SQL—ensuring data integrity for accurate analysis.

Exploratory Data Analysis (EDA)

I uncover trends, correlations, and patterns using descriptive statistics, data profiling, and visualizations through Seaborn, Matplotlib, and Plotly.

Data Visualization & Reporting

I build interactive dashboards and insightful reports using Power BI, Tableau, and Excel to support decision-making and storytelling.

Dashboard Customization for Stakeholders

Build role-specific dashboards (e.g., for sales, marketing, HR) to ensure each team sees the most relevant KPIs at a glance.

ETL (Extract, Transform, Load) Pipelines

Create and manage automated pipelines to gather, clean, and push data to BI tools or databases using Python, Airflow, or Power Automate.
